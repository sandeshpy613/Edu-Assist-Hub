Method URL Process
GET / Renders the index page (login/signup)
GET /Project Renders the Project page
GET /profile Fetches and renders user profile by std_id
GET /profilestart Renders the profile start page
GET /history Fetches and renders history log
GET /contactus Renders the contact us page
GET /code Renders the code generation page
GET /mcq Renders the MCQ generation page
GET /email Renders the email template page
GET /letter Renders the letter generation page
GET /essay Renders the essay generation page
GET /summary Renders the summary generation page
GET /senti Renders the sentiment analysis page
GET /login Authenticates and renders home page
GET /logout Logs out and renders index page
GET /test-db Tests database connectivity
GET /delete-history Deletes a history entry and redirects
POST /signup Registers a new user (simplified version)
POST /register Registers a new user with full profile
POST /update-profile Updates user profile in database
POST /generate-code Generates code output using AI
POST /generate-mcq Generates MCQs for a given topic
POST /generate-email-template Generates email templates
POST /generate-letter Generates formal letters
POST /generate-essay Generates essays
POST /generate-summary Generates summaries
POST /analyze-sentiment Performs sentiment analysis on text
POST /save-code Saves input/output to history collection
